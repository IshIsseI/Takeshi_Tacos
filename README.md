#　オリジナルゲーム「たけしのタコス屋さん」

## 概要
たけしくんがタコスを作りながら冒険する物語．

敵を倒しタコスの素材を集め，美味しいタコスを作って，待ち受けるボスに挑みます．

ボスは好みのタコスがあるので，試行錯誤しながら攻略を目指します．

## 使用技術
Unity / C#
